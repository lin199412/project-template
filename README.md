#### 基于webpack4制作的demo, 带babel, less编译, webpack热更新, typescript支持, 
#### 页面说明: 在src目录下的view写页面(新页面需要在config补充,自动化待补充), 在src目录下的styles目录下写样式(需要在main.js引入)
```bash
# 安装依赖
yarn
# 启动本地服务
yarn start
# build
yarn run prod
```
